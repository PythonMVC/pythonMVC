## Dev setup
- Fork & branch from `main`
```bash
git clone https://github.com/PythonMVC/pythonMVC
cd pythonMVC
python -m venv .venv && source .venv/bin/activate
pip install -e ".[dev]"    # add a [project.optional-dependencies.dev] if you like
```
